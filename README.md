# PartnerMassage

A calm, no-ads guided massage you can give your partner tonight.

→ Live at **[partnermassage.app](https://partnermassage.app)**

## What it does

PartnerMassage walks an amateur through giving a 60-minute (or 30-minute) back-of-body massage to their partner. Painterly illustrations show the technique, an in-session timer auto-advances through 15 steps from scalp to feet, and animated hand overlays show the motion.

Built for couples — not professional therapists.

## Run it locally

Vanilla HTML/CSS/JS — no framework, no build step, no backend.

```bash
python -m http.server 8000
```

Then visit [http://localhost:8000](http://localhost:8000).
